Turn your post into products or add multiple products into your posts.
== Description ==
Easy to use and customize e-commerce solution. Allows including products labels anywhere on your posts or pages, and use your content as description or details. 

[Plugin Page](http://wordpress-checkout.com/?utm_source=wp_repo_link&utm_medium=options&utm_campaign=wpckt&utm_content=plugin_home_page) | [How To Use](http://wordpress-checkout.com/how-to-use?utm_source=wp_repo_link&utm_medium=options&utm_campaign=wpckt&utm_content=how_to_use) | [Live Demo](http://wordpress-checkout.com/demo?utm_source=wp_repo_link&utm_medium=options&utm_campaign=wpckt&utm_content=demo) |

Special Features:

*   Allows managing post like products or using a simple post for several products
*   Based on shortcodes
*   Uses PayPal Payments Standard by default to process the payments
*   Support the most used currencies 
*   Order report included
*   Transparent layout that match any templates
*   Allows variable shipping depending on weight and region based on PayPal shipping calculator

== Installation ==

1. Auto install or upload the entire 'wordpress-checkout' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a checkout page using the WP editor button
4. Go to the WP Checkout settings page
5. Add the required options
6. Setup your PayPal account
7. Start selling


You will find 'Wp Checkout' menu in your WordPress admin panel and the icon in your post editor panel.

For basic usage, you can also have a look at the [Plugin Page](http://wordpress-checkout.com/?utm_source=wp_repo_link&utm_medium=options&utm_campaign=wpckt&utm_content=plugin_home_page).
